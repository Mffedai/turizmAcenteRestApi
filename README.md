
# Tourism Agency Project

This project encompasses an API developed to manage the operations of a tourism agency company. It is an advanced-level project within the Patika+ Full Stack Developer training.

### Technologies Used


- Spring Boot
- Spring Data JPA
- RESTAPI
- PostgreSQL
- Spring Web
- Authentication
